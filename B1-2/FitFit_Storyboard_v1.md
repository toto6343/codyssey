## 서사 구조 (Narrative Structure)

| 단계 | 씬 | 내용 | 감정 흐름 |
|------|-----|------|----------|
| 기 (Setup) | 씬 1–2 | 사이버펑크 도시 세계관 구축 + 주인공 등장(뒷모습) | Mystery / Intrigue |
| 승 (Development) | 씬 3–4 | 주인공의 재킷 클로즈업 → 풀 룩 워킹 공개 | Desire / Attraction |
| 전 (Turn) | 씬 5 | FIT FIT 브랜드 로고 등장 + 주인공 워킹과 로고 합성 | Recognition / Excitement |
| 결 (Resolution) | 씬 6 | 주인공 → AI 데이터 → 브랜드 아이덴티티로 통합 + CTA | Aspiration / Action |

> **컨셉 메모:** 광고 전체를 한 인물이 관통하며, 도시를 걷고 브랜드를 발견해 나가는 과정을 하나의 연속된 여정으로 그립니다. 인물의 얼굴은 씬 초반에는 가려지고(뒷모습·실루엣) 후반으로 갈수록 서서히 드러나며, 마지막에는 인물 자체가 데이터와 브랜드로 치환되는 구조입니다.

## 스토리보드 (Storyboard)

### Scene 01 — 도시 와이드샷 `5초`

**목표:** 도시의 밤 세계관을 한 컷으로 각인 — 브랜드 세계 진입

**화면 구성:** 와이드샷 · 젖은 도심 도로 / 양쪽 네온사인 / 스팀 안개 / 인물 없음 · 텍스트 없음

**내레이션:** *(사운드 온리) 빗소리 + 로우파이 신스 비트 페이드인*

| 도구 | 내용 |
|------|------|
| 이미지 | Midjourney v6 — cyberpunk city keyvisual 생성 |
| 비디오 | Google Flow — 정지 이미지 → 카메라 슬로우 달리인 모션 변환 |
| 오디오 | Suno AI — lofi cyberpunk rain intro BGM 생성 |

**프롬프트:**
```
cyberpunk neon-lit city street, rain reflections on wet asphalt, pink and cyan neon signs,
heavy fog, empty street, cinematic wide shot, 8k, --ar 16:9 --style raw
```

**출력 결과:** 핑크·청록 네온 반사가 살아있는 도시 원경 확보, 안개 질감 충분

---

### Scene 02 — 주인공 등장 (뒷모습) `5초`

**목표:** 주인공을 도시 속 실루엣으로 처음 각인 — 신비감 조성

**화면 구성:** 풀바디 · 도로 중앙 / 주인공 뒷모습(back silhouette) 구도 / 홀로그래픽 재킷 / 네온사인 배경 / 카메라 천천히 줌인

| 도구 | 내용 |
|------|------|
| 이미지 | Midjourney v6 — 뒷모습 인물 + 홀로그래픽 재킷 |
| 비디오 | Google Flow — 슬로우 줌인 모션 |
| 오디오 | Suno AI — BGM 계속 (빗소리 레이어) |

**프롬프트:**
```
hooded person in city at night, cyberpunk style, back silhouette, standing in rainy street,
holographic iridescent jacket, neon reflections, cinematic, shallow DOF, --ar 16:9
```

**출력 결과:** 뒷모습 인물 구도 + 홀로그래픽 재킷 질감 강한 키비주얼 확보, 얼굴이 보이지 않아 신비감 유지

---

### Scene 03 — 재킷 클로즈업 `4초`

**목표:** 제품(재킷)의 텍스처와 디테일을 강조 — 동일 주인공의 상체를 클로즈업으로 전환

**화면 구성:** 미디엄 클로즈업 · 좁은 골목(그래피티 벽, 빗물 고인 바닥) / 씬2와 동일 인물, 재킷 상체 위주 프레이밍 / 홀로그래픽 재질 디테일 강조

**내레이션:** *(사운드) 비트 드롭 + 빗방울 타격음 강조*

| 도구 | 내용 |
|------|------|
| 이미지 | Midjourney v6 — 재킷 클로즈업 (`--cref` 동일 인물 고정) |
| 비디오 | Google Flow — 슬로우 카메라 틸트 + 빗방울 디테일 |
| 오디오 | Suno AI — 비트 드롭 타이밍 맞춤 편집 |

**프롬프트:**
```
medium close-up, same character, holographic iridescent jacket detail, standing in rainy
graffiti alley, neon circuit pattern details, pink cyan light reflections, cinematic,
--ar 16:9 --cref [character_ref]
```

**출력 결과:** 동일 인물 유지 + 재킷 텍스처 디테일 확보 (기획 의도)

---

### Scene 04 — 풀 룩 워킹 `5초`

**목표:** 주인공의 워킹 무브로 제품 전신 착장을 입체적으로 어필, 동시에 배경 로고로 다음 씬 예고

**화면 구성:** 풀바디 · 골목 중앙 / 홀로그래픽 풀셋 착장 (씬2·3과 동일 인물) / 카메라 사이드 트래킹 / 배경 벽면에 "FIT FIT" 텍스트가 글리치 형태로 자연스럽게 노출

**내레이션:** *"Fit the city. Fit yourself"*

| 도구 | 내용 |
|------|------|
| 이미지 | Midjourney v6 — 풀 룩 전신샷 (`--cref` 동일 인물 고정) |
| 비디오 | Google Flow — 사이드 트래킹 워킹 모션 |
| 오디오 | Suno AI — BGM 빌드업 |

**프롬프트:**
```
full body shot, same character walking in holographic iridescent jacket and pants, neon alley,
side tracking camera, "FIT FIT" glitch text visible on background wall, cyberpunk street,
cinematic color grade, --ar 16:9 --cref [character_ref]
```

**출력 결과:** 풀 룩 워킹 샷 + FIT FIT 브랜드 텍스트가 배경에 자연스럽게 합성되어 씬5로의 전환이 매끄러움

---

### Scene 05 — 로고 등장 `4초`

**목표:** 브랜드 로고를 명확히 각인시키고 주인공의 워킹과 자연스럽게 연결

**화면 구성:** 미디엄샷 · 동일 주인공이 걷는 모습 위로 FIT FIT 로고가 글리치 형태로 오버레이 · 단일 컷 구성

**내레이션:** *"FIT FIT"*

| 도구 | 내용 |
|------|------|
| 이미지 | Midjourney v6 — 로고+인물 워킹 합성 키비주얼 |
| 비디오 | Google Flow — 글리치 트랜지션 효과 |
| 오디오 | Suno AI — BGM 클라이막스 |

**프롬프트:**
```
cyberpunk character walking, same character, large FIT FIT holographic logo projected on
building wall, neon pink cyan, cinematic, --ar 16:9 --cref [character_ref]
```

**출력 결과:** 로고와 주인공이 한 프레임 안에서 자연스럽게 결합, 브랜드 인지 강화

---

### Scene 06 — 아웃트로 + CTA `7초`

**목표:** 주인공에서 브랜드로 시선 이동 — AI/데이터 시각화 트랜지션 → 브랜드 최종 아이덴티티 각인 + CTA

**화면 구성:** 주인공 실루엣 → 컬러 데이터 파티클(핑크/시안/옐로우) 폭발 → 의상 클로즈업 → FIT FIT 로고 최종 고정 → 슬로건 텍스트

| 도구 | 내용 |
|------|------|
| 이미지 | Midjourney v6 — 디지털 뉴럴 네트워크 + 의상 합성 |
| 비디오 | Google Flow — 파티클 폭발 → 재킷 클로즈업 트랜지션 |
| 오디오 | Suno AI (아웃트로) + ElevenLabs TTS (내레이션) |

**프롬프트:**
```
abstract digital neural network morphing into cyberpunk leather jacket, neon data particles
0 1 binary rain, pink cyan yellow, futuristic --ar 16:9

Final: FIT FIT neon logo, tagline text "Fit the city. Fit yourself.", dark sci-fi interior
```

**출력 결과:** 주인공이 데이터로 치환되고 파티클→재킷 트랜지션 + 최종 로고+슬로건 프레임으로 마무리, "결(Resolution)" 단계 완성

## 제약 사항 및 리스크 대응 전략 (Risk Mitigation)

| 발생 이슈 | 적용 전략 |
|----------|----------|
| 인물 일관성 미유지 위험 | Midjourney `--cref` (캐릭터 레퍼런스) 파라미터로 전 씬 동일 인물 고정 |
| 의상 스타일 불일치 | `--sref` 병행 적용 검토, 재킷·색감 디테일을 프롬프트에 명시적으로 반복 기재 |
| 해상도 · 비율 혼재 | 모든 이미지 생성 시 `--ar 16:9` 고정, Google Flow 출력 해상도 1080p 통일 |
| 오디오 톤 미스매치 | Suno AI 단일 세션에서 BGM 전체 생성 후 씬별 컷 편집 (동일 세션 생성으로 톤 일관성 확보) |
| 크레딧 / 대기열 이슈 | 이미지 스토리보드 완전 확정 후 영상 변환 진행, 씬 4개 우선 제작 후 단계적 확장 |
| 도구 접근 제한 | 비디오 대체: Pika 2.0 / Kling \| 오디오 대체: Udio \| TTS 대체: Play.ht |
