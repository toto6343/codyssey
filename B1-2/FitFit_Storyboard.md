## 서사 구조 (Narrative Structure)

| 단계 | 씬 | 내용 | 감정 흐름 |
|------|-----|------|----------|
| 기 (Setup) | 씬 1–2 | 사이버펑크 도시 세계관 구축 + 미지의 인물 등장 | Mystery / Intrigue |
| 승 (Development) | 씬 3–4 | 의상 디테일 클로즈업 + 풀 룩 워킹 공개 | Desire / Attraction |
| 전 (Turn) | 씬 5 | FIT FIT 브랜드 로고 글리치 등장 | Recognition / Excitement |
| 결 (Resolution) | 씬 6 | AI 데이터 → 브랜드 아이덴티티 통합 + CTA | Aspiration / Action |

## 스토리보드 (Storyboard)

### Scene 01 — 도시 와이드샷 `5초`

**목표:** 도시의 밤 세계관을 한 컷으로 각인 — 브랜드 세계 진입

**화면 구성:** 와이드샷 · 젖은 도심 도로 / 양쪽 네온사인 / 스팀 안개 / 인물 없음 · 텍스트 없음

**내레이션:** *(사운드 온리) 빗소리 + 로우파이 신스 비트 페이드인*

| 도구 | 내용 |
|------|------|
| 이미지 | Midjourney v6 — cyberpunk city keyvisual 생성 |
| 비디오 | Google Flow — 정지 이미지 → 카메라 슬로우 달리인 모션 변환 |
| 오디오 | Suno AI — lofi cyberpunk rain intro BGM 생성 |

**프롬프트:**
```
cyberpunk neon-lit city street, rain reflections on wet asphalt, pink and cyan neon signs,
heavy fog, cinematic wide shot, no people, 8k, --ar 16:9 --style raw
```

**출력 결과:** 핑크·청록 네온 반사가 살아있는 도시 원경 확보, 안개 질감 충분

---

### Scene 02 — 인물 실루엣 등장 `5초`

**목표:** 후드 인물 실루엣 등장 — 브랜드 주인공 첫 인지

**화면 구성:** 풀바디 · 도로 중앙 / 후드 인물 실루엣 / 네온사인 배경 / 카메라 천천히 줌인

**내레이션:** *"밤이 내 핏을 정의한다."*

| 도구 | 내용 |
|------|------|
| 이미지 | Midjourney v6 — 후드 인물 실루엣 + 홀로그래픽 재킷 |
| 비디오 | Google Flow — 슬로우 줌인 모션 |
| 오디오 | Suno AI — BGM 계속 (빗소리 레이어) |

**프롬프트:**
```
hooded figure standing in rainy cyberpunk street, center frame, holographic iridescent jacket,
neon reflections, back silhouette, cinematic, shallow DOF, --ar 16:9
```

**출력 결과:** 후드 실루엣 + 홀로그래픽 재킷 질감 강한 키비주얼 확보

#### 프롬프트 수정 로그

| | 프롬프트 | 이슈 |
|--|---------|------|
| 수정 전 | `hooded person in city at night, cyberpunk style` | 인물 얼굴 디테일이 과도하게 표현되어 브랜드 톤과 불일치, 의상보다 인물에 시선 집중 |
| 수정 후 | `hooded figure standing in rainy cyberpunk street, back silhouette, holographic iridescent jacket` | 실루엣 중심으로 전환, 의상 텍스처 강조 |

**수정 이유:** 얼굴 표정이 과도하게 드러나 신비감이 사라졌고, 의상보다 인물에 시선이 쏠림. 실루엣 + 의상 텍스처 중심으로 재조정하여 브랜드 분위기 일관성 확보.

---

### Scene 03 — 재킷 클로즈업 `4초`

**목표:** 의상 디테일 클로즈업 — 제품(재킷) 핵심 어필

**화면 구성:** 미디엄 클로즈업 · 재킷 상체 / 홀로그래픽 텍스처 / 네온 회로 패턴 / 빗방울 슬로모

**내레이션:** *(사운드) 비트 드롭 + 빗방울 타격음 강조*

| 도구 | 내용 |
|------|------|
| 이미지 | Midjourney v6 — 재킷 텍스처 클로즈업 (`--cref` 동일 캐릭터 유지) |
| 비디오 | Google Flow — 슬로우모션 빗방울 + 카메라 틸트업 |
| 오디오 | Suno AI — 비트 드롭 타이밍 맞춤 편집 |

**프롬프트:**
```
close-up of holographic iridescent jacket with neon circuit patterns, rain drops in slow motion,
pink cyan light reflections, ultra detailed fabric texture, product shot, --ar 16:9 --cref [scene02_ref]
```

**출력 결과:** 홀로그래픽 소재 질감 + 회로 패턴 선명하게 표현, 의상 상품성 강조

---

### Scene 04 — 풀 룩 워킹 `5초`

**목표:** 풀 룩 공개 + 워킹 무브 — 제품 전신 착장 어필

**화면 구성:** 풀바디 · 골목 중앙 / 홀로그래픽 풀셋 착장 / 카메라 사이드 트래킹

**내레이션:** *"핏이 나를 말한다."*

| 도구 | 내용 |
|------|------|
| 이미지 | Midjourney v6 — 풀 룩 전신샷 (`--sref` 스타일 고정) |
| 비디오 | Google Flow — 사이드 트래킹 워킹 모션 |
| 오디오 | Suno AI — BGM 빌드업 |

**프롬프트:**
```
full body shot, person walking in holographic iridescent jacket and pants, neon alley,
side tracking camera, cyberpunk street, cinematic color grade, --ar 16:9 --sref [style_ref]
```

**출력 결과:** 풀 룩 워킹 샷 + FIT FIT 브랜드 텍스트 배경 자연스럽게 합성됨

| 비디오 | Google Flow — 글리치 트랜지션 효과 |
| 오디오 | Suno AI — BGM 클라이막스 |

**프롬프트:**
```
cyberpunk woman walking, large FIT FIT holographic logo projected on building wall,
neon pink cyan, cinematic, --ar 16:9
```

**출력 결과:** 인물+로고 배경 자연스러운 합성, 네온 글리치 효과 확보

### Scene 06 — 아웃트로 + CTA `7초`

**목표:** AI/데이터 시각화 트랜지션 → 브랜드 최종 아이덴티티 각인 + CTA

**화면 구성:** 추상 디지털 → 의상 클로즈업 / 회로+데이터 파티클 / FIT FIT 로고 최종 고정 / FITFIT.AI 텍스트

**내레이션:** *"데이터가 디자인하다. 핏이 완성하다." / (화면) FITFIT.AI*

| 도구 | 내용 |
|------|------|
| 이미지 | Midjourney v6 — 디지털 뉴럴 네트워크 + 의상 합성 |
| 비디오 | Google Flow — 파티클 폭발 → 재킷 클로즈업 트랜지션 |
| 오디오 | Suno AI (아웃트로) + ElevenLabs TTS (내레이션) |

**프롬프트:**
```
abstract digital neural network morphing into cyberpunk leather jacket, neon data particles
0 1 binary rain, pink cyan yellow, futuristic --ar 16:9

Final: FIT FIT neon logo, tagline text, dark sci-fi interior
```

**출력 결과:** 파티클→재킷 트랜지션 + 최종 로고+슬로건 프레임 확보, CTA 텍스트 포함

## 제약 사항 및 리스크 대응 전략 (Risk Mitigation)

| 발생 이슈 | 적용 전략 |
|----------|----------|
| 캐릭터 / 스타일 불일치 | Midjourney `--cref` (캐릭터 레퍼런스) 및 `--sref` (스타일 레퍼런스) 파라미터 고정으로 씬 간 일관성 유지 |
| 해상도 · 비율 혼재 | 모든 이미지 생성 시 `--ar 16:9` 고정, Google Flow 출력 해상도 1080p 통일 |
| 오디오 톤 미스매치 | Suno AI 단일 세션에서 BGM 전체 생성 후 씬별 컷 편집 (동일 세션 생성으로 톤 일관성 확보) |
| 크레딧 / 대기열 이슈 | 이미지 스토리보드 완전 확정 후 영상 변환 진행, 씬 4개 우선 제작 후 단계적 확장 |
| 도구 접근 제한 | 비디오 대체: Pika 2.0 / Kling \| 오디오 대체: Udio \| TTS 대체: Play.ht |
