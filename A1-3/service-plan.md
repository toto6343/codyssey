# FitRoutine (핏루틴) — 서비스 기획서

## 1. 서비스 개요

- **서비스명**: FitRoutine (핏루틴)
- **목적**: 헬스장에 가지 않고도 집에서 바로 시작할 수 있는 맞춤 운동 루틴을 AI가 즉시 생성해주는 서비스
- **타겟 사용자**: 헬스장 등록이 부담스럽거나 시간이 애매하게 남는 직장인/학생 — "지금 당장 15분만 운동하고 싶은 사람"

## 2. 페이지/섹션 구성 (네비게이션으로 이동)

| 섹션 | 내용 |
|---|---|
| **홈 (Hero)** | 서비스 한 줄 소개 + 바로 루틴 생성 섹션으로 이동하는 CTA |
| **루틴 생성기** | 운동 목표 + 가능 시간을 선택하면 AI가 루틴을 생성해 화면에 표시 |
| **운동 팁 / FAQ** | 운동 전후 주의사항, 서비스 사용법 안내 |

## 3. 핵심 기능

1. 운동 목표(근력/유산소/스트레칭) + 시간(15/30/60분) 선택 UI
2. AI가 선택 조건에 맞는 순서형 운동 루틴(운동명 + 세트/시간) 생성
3. 생성 결과를 카드 형태로 화면에 표시
4. 실패 상황 안내 (아래 4번 참고)

## 4. AI 기능 설계

| 항목 | 내용 |
|---|---|
| **입력** | 운동 목표(문자열, 예: "근력"), 가능 시간(문자열, 예: "30분") |
| **출력** | 순서가 있는 운동 목록 텍스트 (운동명 - 세트/횟수 또는 시간) |
| **처리 방식** | 프론트에서 `fetch('/api/generate')`로 POST 요청 → Vercel Serverless Function(Python)이 OpenAI API 호출 → 결과를 JSON으로 반환 |
| **실패 처리 - 빈 입력** | 목표 또는 시간을 선택하지 않고 제출 시, API 호출 없이 "운동 목표와 시간을 모두 선택해주세요" 안내 표시 |
| **실패 처리 - API 오류(4xx/5xx)** | "루틴 생성 중 문제가 발생했습니다. 잠시 후 다시 시도해주세요" 안내 표시 |
| **실패 처리 - 지연/타임아웃** | 15초 이상 응답이 없으면 요청을 중단하고 "응답이 지연되고 있습니다. 다시 시도해주세요" 안내 표시 |

## 5. 기술 스택

- Frontend: 바닐라 HTML/CSS/JavaScript (프레임워크 미사용)
- Backend: Vercel Serverless Functions (Python, 표준 라이브러리 `urllib` 사용 — 외부 패키지 불필요)
- AI: OpenAI API (Chat Completions)
- 배포: Vercel (GitHub 연동)

## 6. 반응형 기준

- 데스크톱(1200px 이상), 태블릿(768px), 모바일(375px 내외) 3개 화면 크기에서 레이아웃 확인
- 768px 이하에서는 상단 네비게이션이 햄버거 메뉴로 전환
