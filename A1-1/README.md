# 나만의 프롬프트 관리 프로그램

GenAI를 활용하며 쌓아온 프롬프트들을 흩어진 메모장 대신 한 곳에서
체계적으로 관리하기 위해 만든 파이썬 콘솔 프로그램입니다.
제목/내용/카테고리로 프롬프트를 등록하고, 검색·즐겨찾기·조회수 관리는
물론 JSON 저장과 Markdown 내보내기까지 지원합니다.

## 실행 방법

```bash
python prompt_manager.py
```

Python 3.10 이상이 필요하며, 외부 라이브러리 설치 없이 표준 라이브러리(`json`, `os`)만 사용합니다.

## 기능 목록

### 기본 기능
1. **프롬프트 추가** – 제목, 내용, 카테고리를 입력해 새 프롬프트를 등록합니다. (빈 값 입력 방지)
2. **프롬프트 목록** – 전체 프롬프트를 번호, 카테고리, 즐겨찾기 여부와 함께 출력합니다.
3. **카테고리별 조회** – 원하는 카테고리를 선택해 해당 프롬프트만 모아봅니다.
4. **프롬프트 검색** – 제목 또는 내용에 포함된 키워드로 검색합니다.
5. **프롬프트 상세 보기** – 번호를 입력하면 전체 내용을 확인합니다. (조회 시 조회수 자동 증가)
6. **즐겨찾기 추가/해제** – 자주 쓰는 프롬프트를 즐겨찾기로 표시하거나 해제합니다.
7. **즐겨찾기 목록** – 즐겨찾기한 프롬프트만 모아봅니다.

### 보너스 기능
8. **프롬프트 수정** – 기존 프롬프트의 제목/내용/카테고리를 수정합니다. (CRUD)
9. **프롬프트 삭제** – 더 이상 필요 없는 프롬프트를 삭제합니다. (CRUD)
10. **조회수 Top 목록** – 상세 보기로 확인한 횟수를 기준으로 프롬프트를 정렬해 보여줍니다. (사용 기록)
11. **JSON 파일로 저장** – 현재 프롬프트 데이터를 `prompts.json`으로 저장해 다음 실행 시 이어서 사용할 수 있습니다. (영속화)
12. **JSON 파일 불러오기** – 저장해둔 `prompts.json`을 불러옵니다. (영속화)
13. **카테고리별 Markdown 내보내기** – 전체 프롬프트를 카테고리별 `.md` 파일로 `exports/` 폴더에 내보냅니다. (내보내기)

## 등록된 프롬프트 카테고리

- 텍스트 생성
- 이미지 생성
- 영상 생성
- 페르소나
- 자동화
- 기타 (직접 입력 가능)

## 기본 데이터

프로그램 최초 실행 시 아래 4개의 프롬프트가 기본으로 등록되어 있습니다.

1. [텍스트 생성] 블로그 글 작성 도우미 ⭐
2. [이미지 생성] 제품 썸네일 생성
3. [페르소나] IT 컨설턴트 페르소나
4. [자동화] 뉴스 요약 프롬프트

## 코드 구조

모든 기능은 역할별 함수로 분리되어 있습니다. (`add_prompt`, `show_list`,
`search_prompt`, `toggle_favorite`, `save_to_json` 등) 프로그램 실행 중
추가/수정/삭제된 데이터는 메모리(리스트)에 유지되며, 종료 시 초기화됩니다.
필요하다면 11번 기능으로 JSON에 저장해 다음 실행에 이어서 사용할 수 있습니다.
