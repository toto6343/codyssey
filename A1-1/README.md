# 나만의 프롬프트 도서관 (Prompt Library)

파편처럼 흩어져 있던 AI 프롬프트를, "도서관"에 비유해서 체계적으로 등록·검색·관리하는
파이썬(Python) 콘솔 프로그램입니다.

- 프롬프트 = **도서**
- 카테고리 = **서가(분야)**
- 즐겨찾기 = **추천 도서**

## 실행 방법

1. Python 3.10 이상이 설치되어 있어야 합니다.
   ```bash
   python3 --version
   ```
2. 저장소를 내려받은 뒤 프로그램을 실행합니다.
   ```bash
   git clone https://github.com/toto6343/codyssey.git
   cd A1-1
   python3 prompt_library.py
   ```
3. 실행 후 나타나는 메뉴에서 원하는 기능 번호를 입력하면 됩니다.

## 기능 목록

### 필수 기능
| 번호 | 메뉴 | 설명 |
|---|---|---|
| 1 | 신간 등록 | 새 프롬프트(도서)를 등록합니다. 제목/내용은 빈 값이면 다시 입력받습니다. |
| 2 | 전체 서가 보기 | 등록된 모든 프롬프트를 번호·카테고리·즐겨찾기 표시(⭐)와 함께 출력합니다. |
| 3 | 분야별 서가 | 카테고리를 선택하면 해당 카테고리의 프롬프트만 조회합니다. |
| 4 | 도서 검색 | 키워드가 제목 또는 내용에 포함된 프롬프트를 검색합니다. |
| 5 | 도서 상세 열람 | 번호를 입력하면 제목/카테고리/즐겨찾기/내용 전체를 보여줍니다. |
| 6 | 추천 도서 등록/해제 | 즐겨찾기를 토글(추가/해제)합니다. |
| 7 | 추천 도서 서가 | 즐겨찾기한 프롬프트만 모아서 봅니다. |
| 0 | 종료 | 프로그램을 종료합니다. |

### 보너스 1 — 영속화 및 내보내기
| 번호 | 메뉴 | 설명 |
|---|---|---|
| 11 | 도서관 저장하기 | 전체 프롬프트를 `library_data.json`으로 저장합니다. |
| 12 | 도서관 불러오기 | 저장된 JSON 파일에서 프롬프트를 불러옵니다. |
| 13 | 서가별 내보내기 | 카테고리별로 `exports/카테고리명.md` 파일을 생성합니다. |

### 보너스 2 — CRUD 및 사용 기록
| 번호 | 메뉴 | 설명 |
|---|---|---|
| 8 | 도서 정보 수정 | 제목/내용/카테고리를 수정합니다. |
| 9 | 도서 폐기 | 프롬프트를 삭제합니다. |
| 10 | 인기 도서 TOP 목록 | 상세 열람 시 누적되는 조회수 기준으로 상위 항목을 보여줍니다. |

> 실행 중 추가/수정/삭제한 내용과 즐겨찾기 상태는 프로그램을 종료하면 초기화됩니다.
> 계속 보관하고 싶다면 종료 전 `11. 도서관 저장하기`로 JSON에 저장해두세요.

## 등록된 프롬프트 카테고리(서가)

- 텍스트 생성
- 이미지 생성
- 영상 생성
- 페르소나
- 자동화
- 기타

## 코드 구조

모든 기능은 `show_menu`, `add_prompt`, `show_list`, `show_by_category`, `search_prompt`,
`show_detail`, `toggle_favorite`, `show_favorites`, `edit_prompt`, `delete_prompt`,
`show_top_viewed`, `save_to_json`, `load_from_json`, `export_markdown_by_category` 등
기능 단위 함수로 분리되어 있으며, `main()`에서 메뉴 선택에 따라 각 함수를 호출합니다.
