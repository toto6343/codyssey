# AI 서비스 기획 전문가 프롬프트

## 프로젝트 개요

서비스 아이디어를 입력하면 실무 수준의 서비스 기획 문서를 자동 생성하는 AI 프롬프트를 설계하였다.

## 목표

* 아이디어를 빠르게 서비스 기획서 형태로 구조화
* MVP 정의 지원
* 사업성 검토 지원
* 반복 가능한 기획 프로세스 구축

## 프롬프트 설계

Custom Instructions 기능을 활용하여 다음과 같은 역할을 정의하였다.

* IT 서비스 기획 전문가(PM/PO)
* 서비스 개요 작성
* 문제 정의
* 핵심 기능 도출
* 사용자 시나리오 작성
* MVP 정의
* 확장 전략 제안

## 테스트 수행

### 입력

반려동물 건강관리 SaaS 서비스를 기획해줘

### 결과

PetCare+라는 서비스 기획안 생성

주요 내용:

* AI 기반 건강관리 플랫폼
* 건강기록 통합 관리
* 예방접종 일정 관리
* AI 건강 분석 기능
* SaaS 구독 모델 제안

## 재사용성

Custom Instructions에 저장하여 새로운 채팅에서도 동일한 형식의 서비스 기획서를 반복 생성할 수 있다.

## 활용 가능 분야

* 스타트업 아이디어 검증
* 신규 서비스 기획
* 사업계획서 초안 작성
* PRD 작성 지원
* MVP 설계

## 결론

Custom Instructions를 활용하여 서비스 기획 업무를 지원하는 재사용 가능한 AI 프롬프트를 구축하였다.
