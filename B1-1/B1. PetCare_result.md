# PetCare+ 서비스 기획 프로젝트

## 프로젝트 개요

PetCare+는 AI 기반 반려동물 건강관리 및 예방 케어 SaaS 플랫폼입니다.

반려동물의 건강기록, 예방접종, 운동량, 건강 상태를 통합 관리하고, AI 분석을 통해 질병 위험을 조기에 탐지하여 예방 중심의 헬스케어 서비스를 제공합니다.

PetCare+

### 서비스 한 줄 소개

AI 기반 반려동물 건강관리 및 예방 케어 SaaS 플랫폼

* 의료비 절감

## 문제 정의

현재 반려동물 보호자들은 예방접종, 진료기록, 약 복용 이력 등을 개별적으로 관리하고 있으며 병원마다 데이터가 분산되어 있습니다.

또한 건강 상태를 지속적으로 모니터링하기 어렵기 때문에 질병을 조기에 발견하지 못하는 문제가 발생합니다.

### 기존 방식의 한계

| 구분      | 한계            |
| ------- | ------------- |
| 수기 건강수첩 | 기록 누락 및 관리 불편 |
| 병원 앱    | 특정 병원에 종속     |
| 스마트 기기  | 데이터 분석 기능 부족  |
